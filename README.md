# 簡易佈告欄系統

前後端分離的教學示範專案。  
前端以 Vite + React + TypeScript 建構，後端以 Google Apps Script + Google Sheets 提供 API。

## 專案結構

```
BulletinBoard/
├── frontend/   # React 前端，部署至 GitHub Pages
├── backend/    # Google Apps Script，部署至 GAS Web App
├── AGENTS.md   # AI 協作說明
└── README.md
```

詳細說明請見各子目錄的 README：

- [frontend/README.md](frontend/README.md)
- [backend/README.md](backend/README.md)
